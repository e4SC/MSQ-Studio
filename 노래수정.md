# 노래 수정 / 노래 리스트 사용방법

이 문서는 **두 화면(노래 수정, 노래 리스트)** 을 한 번에 설명합니다.  

---

## 준비물

- 실행 폴더에 `Setting.py` 파일이 있어야 합니다.
- `Setting.py`가 없으면 목록/수정/삭제 기능이 동작하지 않습니다.

---

# 1) 노래 수정 화면

이 화면에서는 아래 작업을 할 수 있습니다.

- **번호로 특정 곡을 편집 화면으로 열기**
- **두 곡의 순서(번호) 서로 교환**
- **특정 곡 삭제**
- **전체 삭제**
- 모든 작업은 실행 전에 **미리보기**로 대상 확인 가능

---

## 1-1. 번호로 편집 열기

1. 수정할 **곡 번호**를 입력합니다.
2. 아래 **미리보기**에 해당 번호의 정보가 표시되는지 확인합니다.
3. 버튼을 누르거나 **Enter**를 누르면 편집 화면이 열립니다.

✅ 체크 포인트  
- 미리보기에 `(없음)`으로 뜨면 그 번호는 존재하지 않습니다.

---

## 1-2. 순서 변경(두 번호 교환)

1. 교환할 **두 개의 번호(A, B)** 를 입력합니다.
2. 미리보기에서 A/B가 각각 어떤 곡인지 확인합니다.
3. 실행 버튼을 누르면 **변경 전/후** 비교가 뜹니다.
4. 확인에서 Yes를 누르면 실제로 교환됩니다.

⚠️ 주의  
- 교환은 “번호 자체의 의미”가 바뀌는 작업이라 운영 중이면 실수 영향이 큽니다.  
  반드시 미리보기와 확인창을 보고 진행하세요.

---

## 1-3. 노래 리스트 열기

- 노래 리스트 열기 버튼을 누르면 **전체 곡 리스트 화면**이 열립니다.
- 목록 화면에서도 더블클릭/Enter로 바로 편집 화면을 열 수 있습니다.

---

## 1-4. 특정 곡 삭제

1. 삭제할 **곡 번호**를 입력합니다.
2. 미리보기로 삭제 대상이 맞는지 확인합니다.
3. 실행 버튼을 누르면 확인창이 뜹니다.
4. Yes를 누르면 해당 곡이 삭제됩니다.

⚠️ 주의  
- 삭제는 되돌리기 어렵습니다. 실행 전 백업 권장.

---

## 1-5. 전체 삭제

- 전체 삭제 버튼을 누르면 `Setting.py` 내용이 **완전히 비워집니다.**
- 확인에서 Yes를 누르면 즉시 적용됩니다.

🚨 매우 중요  
- 테스트가 아니라면 정말 신중하게 사용하세요.

---

## 1-6. 입력/편의 기능

- 번호 입력 칸은 **숫자만** 입력됩니다.
- **Enter**로 즉시 실행됩니다.
- 입력 칸 우클릭 메뉴에서
  - **클립보드에서 숫자만 붙여넣기**
  - 붙여넣기 / 지우기
  같은 기능을 사용할 수 있습니다.

---

# 2) 노래 리스트 화면

이 화면에서는 `Setting.py`의 곡들을 **번호/노래/가수**로 목록화해서 보여줍니다.

- 검색(제목/가수/전체)
- 공백/특수문자 무시 검색
- 중복만 보기
- 더블클릭/Enter로 편집 열기
- 파일 변경 시 자동 갱신

---

## 2-1. 목록 확인

- 목록에는 보통 다음이 표시됩니다.
  - 번호
  - 노래
  - 가수

---

## 2-2. 검색

1. 검색창에 키워드를 입력합니다.
2. 입력 즉시 목록이 필터링됩니다.

### 검색 범위
- **전체**: 노래 + 가수에서 검색
- **노래만**: 노래에서만 검색
- **가수만**: 가수에서만 검색

### 공백/특수문자 무시
- 체크하면 띄어쓰기/특수문자 차이로 검색이 실패하는 경우를 줄여줍니다.  
  예) `A B C` / `A-B-C` / `ABC` 같은 케이스

---

## 2-3. 중복만 보기 (색상 표시)

- “중복만 보기”를 체크하면 **중복으로 판단된 곡만** 표시됩니다.
- 중복 그룹은 **그룹별로 서로 다른 색상**으로 표시될 수 있습니다.
  - 같은 그룹(같은 제목+가수)은 같은 색
  - 다른 그룹은 다른 색

✅ 참고  
- 중복 판단은 보통 “공백/특수문자 제거 후 제목+가수” 기준으로 묶입니다.

---

## 2-4. 편집 화면 열기

- 목록에서 항목 선택 후
  - **더블클릭** 또는
  - **Enter**
- 해당 곡의 편집 화면이 열립니다.

편집을 마치고 돌아오면, 방금 편집했던 곡이 자동으로 다시 선택되도록 동작합니다.

---

## 2-5. 단축키

| 키 | 동작 |
|---|---|
| Ctrl + F | 검색창으로 이동/전체선택 |
| Enter | 선택한 곡 편집 열기 |
| Esc | 닫기 |

---

## 2-6. 자동 갱신(파일 변경 감지)

- `Setting.py`가 저장되거나 변경되면 목록이 자동으로 다시 불러와집니다.
- 저장 중 파일이 잠깐 잠길 수 있는 상황을 대비해서 **짧게 재시도** 후 다시 불러옵니다.
