# 🎵 노래 수정 / 노래 리스트 사용법 (초보자용)

이 문서는 **노래 수정 화면**과 **노래 리스트 화면**을 한 번에 설명하는 안내서입니다.  
두 화면은 서로 연결되어 있으며, 보통 아래 흐름으로 사용합니다.

- **노래 수정 화면**에서 번호로 직접 수정 / 순서 변경 / 삭제
- **노래 리스트 화면**에서 전체 목록을 보고 검색 / 중복 확인 / 편집 열기 / 삭제 / 순서 이동

---

## 1) 실행 전 준비

## 반드시 필요한 파일

### `Setting.py`
이 파일이 있어야 아래 기능이 동작합니다.

- 노래 수정
- 노래 순서 변경
- 특정 곡 삭제
- 전체 삭제
- 노래 리스트 표시
- 검색
- 리스트에서 편집 열기
- 리스트에서 삭제 / 순서 이동

즉, `Setting.py`가 없으면 이 두 화면은 사실상 제대로 사용할 수 없습니다.

---

## 2) 두 화면의 관계

이 두 화면은 따로 노는 화면이 아니라 연결되어 있습니다.

### 노래 수정 화면
이 화면은 **직접 번호를 입력해서 작업하는 화면**입니다.

여기서 할 수 있는 일:
- 번호로 수정 열기
- 두 번호 순서 교환
- 특정 번호 삭제
- 전체 삭제
- 전체 리스트 열기

---

### 노래 리스트 화면
이 화면은 **전체 곡을 한눈에 보고 찾는 화면**입니다.

여기서 할 수 있는 일:
- 전체 목록 보기
- 검색
- 중복만 보기
- 더블클릭 / Enter로 수정 열기
- Delete로 삭제
- 드래그 앤 드롭으로 순서 이동

---

## 3) 노래 수정 화면에서 할 수 있는 것

노래 수정 화면은 아래 네 가지가 핵심입니다.

- 특정 번호의 곡을 수정 화면으로 열기
- 두 곡 번호를 서로 교환
- 특정 곡 삭제
- 전체 곡 삭제

그리고 이 모든 작업 전에 **미리보기**로 대상을 확인할 수 있습니다.

---

# 1. 노래 수정 화면

## 1-1) 화면 구성

노래 수정 화면은 카드 형태로 구성됩니다.

보통 아래 카드들이 보입니다.

- 번호로 수정
- 순서 변경
- 노래 리스트
- 삭제

상단에는 다음 버튼도 있습니다.

- FAQ
- 도움말

환경에 따라 보조 기능 버튼이 더 있을 수도 있습니다.

---

## 1-2) 번호로 수정

이 기능은 **특정 곡 번호를 직접 입력해서 수정 화면을 여는 기능**입니다.

### 사용 방법
1. `노래 번호` 칸에 번호를 입력합니다.
2. 아래 `미리보기`를 확인합니다.
3. `노래 수정` 버튼을 누르거나 `Enter`를 누릅니다.

---

### 미리보기에서 보이는 것
입력한 번호가 존재하면 미리보기에 보통 다음이 뜹니다.

- 곡 번호
- 제목
- 노래
- 가수

예:
- `12번 · 제목 / 노래 / 가수`

번호가 없으면:
- `(없음)` 으로 표시됩니다.

---

### 이 기능의 장점
- 번호만 알면 바로 열 수 있습니다.
- 잘못된 번호를 넣었는지 미리보기로 바로 확인할 수 있습니다.
- 수정 전에 다른 곡을 여는 실수를 줄일 수 있습니다.

---

## 1-3) 순서 변경

이 기능은 **두 곡 번호를 서로 교환**하는 기능입니다.

즉:
- A에 있던 곡은 B 위치로
- B에 있던 곡은 A 위치로

바뀝니다.

---

### 사용 방법
1. A 칸에 첫 번째 번호를 입력합니다.
2. B 칸에 두 번째 번호를 입력합니다.
3. 아래 미리보기에서 각각 어떤 곡인지 확인합니다.
4. `순서 변경` 버튼을 누릅니다.
5. 확인창에서 변경 전 / 변경 후를 보고 Yes를 누르면 적용됩니다.

---

### 순서 변경 전에 꼭 확인할 것
이 기능은 단순 이름 바꾸기가 아니라  
**곡 블록 자체의 순서가 바뀌는 작업**입니다.

그래서:
- 운영 중인 데이터
- 이미 번호를 기준으로 쓰고 있는 자료
- 다른 곳과 연결된 순서

가 있다면 영향이 있을 수 있습니다.

반드시 미리보기와 확인창을 보고 진행하는 것이 좋습니다.

---

### 미리보기에서 보이는 것
A와 B 각각에 대해:

- 번호
- 제목
- 노래
- 가수

가 표시됩니다.

둘 다 번호가 맞는지 꼭 확인한 뒤 진행하세요.

---

## 1-4) 노래 리스트 열기

이 버튼은 **노래 리스트 화면**을 여는 기능입니다.

이 기능이 필요한 이유는:
- 번호를 모를 때
- 전체 목록을 보면서 찾고 싶을 때
- 검색해서 바로 수정하고 싶을 때
- 중복을 확인하고 싶을 때

입니다.

즉, 번호를 이미 알고 있으면 수정 화면만으로도 충분하지만,  
번호를 모르거나 전체를 봐야 할 때는 리스트 화면이 더 편합니다.

---

## 1-5) 특정 곡 삭제

이 기능은 **입력한 번호의 곡 하나만 삭제**하는 기능입니다.

### 사용 방법
1. 삭제할 번호를 입력합니다.
2. 미리보기로 대상이 맞는지 확인합니다.
3. `노래 제거` 버튼을 누릅니다.
4. 확인창에서 Yes를 누르면 삭제됩니다.

---

### 삭제 전 확인되는 내용
삭제 확인창에는 보통:
- 번호
- 제목
- 노래
- 가수

가 함께 표시됩니다.

즉, 사용자는 삭제 전에 “정말 이 곡이 맞는지” 확인할 수 있습니다.

---

### 주의할 점
삭제는 되돌리기 어렵습니다.  
중요한 데이터라면 먼저 백업해 두는 것이 좋습니다.

---

## 1-6) 전체 삭제

이 기능은 **모든 곡 정보를 한 번에 비우는 기능**입니다.

즉, `Setting.py`의 내용이 전부 비게 됩니다.

### 사용 방법
1. `모두 제거(전체 삭제)` 버튼을 누릅니다.
2. 확인창에서 Yes를 누르면 전체 삭제가 적용됩니다.

---

### 매우 중요한 주의사항
이 기능은 테스트용이 아니라면 매우 신중하게 써야 합니다.

전체 삭제는:
- 한두 곡만 지우는 것이 아니라
- 전체 데이터를 비우는 것

이기 때문에 실수 시 영향이 큽니다.

---

## 1-7) 번호 입력칸 편의 기능

노래 수정 화면의 번호 입력칸은 편하게 쓰도록 되어 있습니다.

### 숫자만 입력 가능
번호 칸에는 숫자만 입력됩니다.

즉:
- 글자
- 특수문자

는 입력되지 않습니다.

---

### Enter로 바로 실행
각 번호 입력칸에서 Enter를 누르면  
상황에 맞는 버튼이 바로 실행됩니다.

예:
- 수정 번호 칸에서 Enter → 수정 열기
- 삭제 번호 칸에서 Enter → 삭제 실행
- A/B 칸에서 Enter → 순서 변경 실행

---

### 우클릭 메뉴
번호 입력칸을 우클릭하면 아래 기능을 사용할 수 있습니다.

- 클립보드에서 숫자만 붙여넣기
- 일반 붙여넣기
- 지우기

특히 **클립보드에서 숫자만 붙여넣기**는  
문장 속에 번호가 섞여 있어도 숫자만 뽑아 넣을 때 편합니다.

예:
- 클립보드 내용이 `노래번호 15번`
- 숫자만 붙여넣기 실행
- 입력칸에는 `15`

처럼 들어갑니다.

---

## 1-8) FAQ / 도움말

상단에는 FAQ / 도움말 버튼이 있습니다.

### FAQ
노래 추가 / 수정 관련 자주 묻는 질문을 엽니다.

### 도움말
노래 수정 설명 문서를 웹페이지로 엽니다.

즉, 작업 중 헷갈릴 때 바로 참고할 수 있습니다.

---

## 1-9) 미리보기의 의미

노래 수정 화면에서 매우 중요한 건 **미리보기**입니다.

미리보기는 단순 장식이 아니라:
- 내가 입력한 번호가 실제로 존재하는지
- 내가 바꾸거나 지우려는 대상이 맞는지

를 확인하기 위한 안전장치입니다.

그래서:
- 수정
- 순서 변경
- 삭제

모두 실행 전에 미리보기를 보는 습관이 좋습니다.

---

## 1-10) 창 닫기와 복귀

노래 수정 화면을 닫으면 보통 메인 화면으로 돌아가도록 되어 있습니다.

즉, 작업이 끝난 뒤 따로 창을 찾아갈 필요 없이  
이전 화면으로 복귀하는 흐름을 생각하면 됩니다.

---

# 2. 노래 리스트 화면

노래 리스트 화면은 **전체 곡을 찾고, 확인하고, 편집하고, 삭제하고, 이동하는 화면**입니다.

---

## 2-1) 화면 구성

노래 리스트 화면은 크게 아래 영역으로 구성됩니다.

- 상단 제목 / 설명
- 새로고침 버튼
- 닫기 버튼
- 검색 카드
- 목록 카드
- 상태 표시줄

---

## 2-2) 검색 카드 구성

검색 카드에는 보통 아래 기능이 있습니다.

- 검색 입력칸
- 공백/특수문자 무시
- 중복만 보기
- 정확히 일치
- 전체 / 노래만 / 가수만

즉, 단순 검색창 하나가 아니라  
검색 방식 자체를 꽤 세밀하게 바꿀 수 있습니다.

---

## 2-3) 목록 카드 구성

목록에는 보통 3개 열이 있습니다.

- 번호
- 노래
- 가수

즉, 사용자는 전체 곡을
“몇 번 곡인지 / 제목이 뭔지 / 가수가 누구인지”
한눈에 볼 수 있습니다.

---

## 2-4) 기본 사용법

### 목록 보기
창을 열면 `Setting.py`의 곡 목록을 읽어서 표시합니다.

---

### 항목 선택
목록에서 원하는 곡을 클릭합니다.

---

### 수정 열기
선택 후 아래 방법 중 하나를 사용하면 수정 화면으로 넘어갑니다.

- 더블클릭
- Enter
- 우클릭 → 수정 열기

---

### 삭제
선택 후 아래 방법 중 하나를 사용하면 삭제할 수 있습니다.

- Delete
- 우클릭 → 삭제

---

## 2-5) 검색 방법

검색창에 단어를 입력하면 목록이 바로 필터링됩니다.

---

### 전체 검색
전체 모드에서는:
- 노래
- 가수
- 번호

를 함께 기준으로 찾습니다.

---

### 노래만 검색
노래명 중심으로 찾습니다.

---

### 가수만 검색
가수명 중심으로 찾습니다.

---

## 2-6) 번호 검색도 가능

검색창에는 문자만이 아니라 번호도 넣을 수 있습니다.

예:
- `15`

이렇게 넣으면 15번 곡을 찾는 데 활용할 수 있습니다.

---

### 번호 여러 개 한 번에 검색
쉼표로 여러 번호를 넣을 수도 있습니다.

예:
- `1,3,11,13`

이렇게 넣으면 해당 번호들만 골라서 볼 수 있습니다.

즉, 특정 번호들만 빠르게 확인할 때 편합니다.

---

## 2-7) 공백 / 특수문자 무시

이 옵션을 켜면 검색할 때:
- 공백
- 특수문자
- 일부 표시용 문자 차이

를 무시하고 찾기 쉽게 해 줍니다.

예를 들어 아래처럼 표기가 조금 달라도 찾기 쉬워집니다.

- `A B C`
- `A-B-C`
- `ABC`

즉, 정확히 같은 띄어쓰기나 기호를 몰라도 검색이 잘 되게 도와줍니다.

---

## 2-8) 정확히 일치

이 옵션을 켜면 “포함 검색”이 아니라 “완전 일치 검색”으로 바뀝니다.

예:
- 끄면: 일부만 들어 있어도 검색됨
- 켜면: 정확히 같은 값만 검색됨

정확한 제목 / 정확한 가수명만 찾고 싶을 때 유용합니다.

---

## 2-9) 중복만 보기

이 옵션을 켜면 **중복으로 판단된 곡만 표시**합니다.

중복 판단은 보통:
- 정규화된 노래명
- 정규화된 가수명

조합을 기준으로 이루어집니다.

즉, 공백이나 특수문자 차이가 조금 있어도  
실질적으로 같은 제목 + 같은 가수면 중복으로 볼 수 있습니다.

---

## 2-10) 중복 색상 표시

중복으로 감지된 곡은 색상으로 구분될 수 있습니다.

특징:
- 같은 중복 그룹은 같은 색
- 다른 중복 그룹은 다른 색

그래서 사용자 입장에서는  
“어떤 곡끼리 서로 같은 그룹인지”를 색으로 쉽게 볼 수 있습니다.

---

## 2-11) 정렬

목록의 열 제목을 클릭하면 정렬할 수 있습니다.

정렬 가능한 기준:
- 번호
- 노래
- 가수

동작:
- 한 번 클릭 → 오름차순
- 다시 클릭 → 내림차순

즉, 번호순 / 제목순 / 가수순으로 원하는 방식으로 볼 수 있습니다.

---

## 2-12) 새로고침

새로고침 버튼을 누르면 목록을 다시 읽습니다.

이 기능은:
- 외부에서 파일을 수정했을 때
- 갱신 상태가 의심될 때
- 다시 읽고 싶을 때

사용하면 됩니다.

---

## 2-13) 자동 갱신

노래 리스트 화면은 `Setting.py`가 바뀌는 것을 감지해서  
자동으로 다시 불러오도록 되어 있습니다.

즉:
- 다른 화면에서 수정 저장
- 혹은 외부 편집기에서 저장

해도 잠시 후 목록이 자동으로 갱신될 수 있습니다.

이 덕분에 목록이 오래된 상태로 남아 있을 가능성이 줄어듭니다.

---

## 2-14) 선택 유지

목록을 갱신하거나 편집 후 돌아왔을 때  
방금 작업하던 곡을 다시 선택해 주려고 시도합니다.

즉, 작업 중 맥락을 잃지 않도록 도와줍니다.

---

## 2-15) 우클릭 메뉴

목록에서 우클릭하면 상황에 따라 메뉴가 나타납니다.

대표 기능:
- 수정 열기
- 삭제

선택 개수에 따라 메뉴 문구도 달라질 수 있습니다.

예:
- 하나 선택 → `삭제 (15번)`
- 여러 개 선택 → `선택 3개 삭제`

---

## 2-16) 여러 곡 삭제

노래 리스트에서는 여러 곡을 동시에 선택해서 삭제할 수 있습니다.

### 사용 방법
1. 여러 곡 선택
2. Delete 또는 우클릭 삭제
3. 확인창 확인
4. Yes 선택

---

### 삭제 전 표시되는 내용
삭제 전에 확인창에서:
- 몇 개를 삭제할지
- 어떤 곡들인지

미리 보여줍니다.

곡이 많으면 일부만 보여주고 나머지는 개수로 표시될 수 있습니다.

즉, 실수 삭제를 줄이기 위한 확인 단계가 있습니다.

---

## 2-17) 드래그 앤 드롭 순서 이동

노래 리스트에서는 곡을 드래그해서 순서를 바꿀 수 있습니다.

이 기능은 **단일 선택일 때만 가능**합니다.

---

### 사용 방법
1. 곡 하나만 선택합니다.
2. 마우스로 끌어서 원하는 위치로 이동합니다.
3. 놓으면 실제 순서가 변경됩니다.

---

### 드래그 중 표시
드래그 중에는:
- 삽입 위치 표시선
- 상태 표시줄 안내

등으로 어디로 이동할지 알려줍니다.

예:
- `↑ 맨 위로 이동`
- `이동 예정 위치: 12번`

---

### 자동 스크롤
드래그 중 목록 위/아래 가장자리로 가져가면  
자동으로 위/아래 스크롤됩니다.

즉, 긴 목록에서도 아래쪽이나 위쪽으로 쉽게 이동할 수 있습니다.

---

### 여러 개 선택 시 드래그
여러 곡을 선택한 상태에서는 드래그 이동이 되지 않습니다.

이 경우:
- 단일선택일 때만 가능하다는 안내가 뜰 수 있습니다.

---

## 2-18) 툴팁

노래 리스트 화면에는 마우스를 올렸을 때 설명이 뜨는 부분이 많습니다.

예:
- 검색창
- 체크박스
- 열 제목
- 목록 항목
- 상태 표시줄

특히 목록 항목 툴팁에는:
- 번호
- 노래
- 가수
- 중복 여부
- 수정/삭제/드래그 가능 안내

등이 나와서 처음 쓰는 사람에게 도움이 됩니다.

---

## 2-19) 상태 표시줄

화면 아래 상태 표시줄에는 보통 다음 정보가 나옵니다.

- 총 곡 수
- 현재 표시 곡 수
- 마지막 갱신 시간

정렬 / 이동 / 드래그 상태에 따라  
안내 문구가 잠시 바뀌기도 합니다.

---

## 2-20) 창 닫기

노래 리스트 창을 닫으면  
보통 이전의 노래 수정 화면으로 돌아가도록 되어 있습니다.

즉, 독립된 창이지만 연결 흐름 안에서 사용된다고 생각하면 됩니다.

---

## 2-21) 위치 / 설정 기억

노래 리스트 화면은 일부 상태를 기억합니다.

예:
- 창 위치
- 검색 옵션 상태
- 중복만 보기 여부
- 공백/특수문자 무시 여부
- 정확히 일치 여부
- 검색 모드

즉, 다음에 열었을 때도 이전에 쓰던 방식에 가깝게 복원됩니다.

---

# 3. 두 화면을 함께 쓰는 추천 방식

초보자라면 아래 흐름이 가장 편합니다.

### 1) 노래 리스트 열기
전체 곡을 먼저 봅니다.

### 2) 검색
제목 / 가수 / 번호로 원하는 곡을 찾습니다.

### 3) 수정 열기
더블클릭 또는 Enter로 해당 곡 수정 화면을 엽니다.

### 4) 수정 후 저장
필요한 수정 작업을 합니다.

### 5) 다시 목록 확인
목록으로 돌아와 결과를 다시 확인합니다.

### 6) 필요하면 순서 이동 / 삭제
리스트나 수정 화면에서 적절한 기능을 사용합니다.

---

# 4. 자주 헷갈리는 부분 정리

### Q. 번호로 수정이 안 열려요
입력한 번호가 실제로 존재하는지 먼저 미리보기에서 확인하세요.  
`(없음)`이면 해당 번호는 없는 것입니다.

---

### Q. 순서 변경은 이름만 바꾸는 건가요?
아니요.  
곡 블록 자체의 위치가 바뀌는 것입니다.

---

### Q. 리스트에서 여러 개 선택하고 드래그할 수 있나요?
아니요.  
드래그 이동은 단일 선택일 때만 가능합니다.

---

### Q. 여러 개 삭제는 가능한가요?
가능합니다.  
리스트 화면에서 여러 항목을 선택한 뒤 Delete 또는 우클릭 삭제를 사용하면 됩니다.

---

### Q. 검색창에 번호 여러 개를 넣을 수 있나요?
가능합니다.  
예: `1,3,11,13`

---

### Q. 검색이 잘 안 될 때는 어떻게 하나요?
아래를 시도해 보세요.

- `공백/특수문자 무시` 켜기
- `전체` 검색 모드로 바꾸기
- `정확히 일치` 끄기

---

### Q. 삭제하면 되돌릴 수 있나요?
쉽게 되돌리기 어렵습니다.  
중요한 데이터면 먼저 백업해 두는 것이 좋습니다.

---

### Q. 전체 삭제는 진짜 전부 지우나요?
네.  
전체 데이터를 비우는 기능이므로 매우 신중하게 사용해야 합니다.

---

# 5. 단축키 요약

## 노래 수정 화면
- `Enter` : 현재 입력칸 기준 실행
- 번호 입력칸에는 숫자만 입력 가능

---

## 노래 리스트 화면
- `Ctrl + F` : 검색창으로 이동
- `Enter` : 선택 곡 수정 열기
- `Delete` : 선택 곡 삭제
- `Esc` : 창 닫기

---

# 6. 꼭 기억하면 좋은 점

- 노래 수정 화면은 **번호 기반 작업용**
- 노래 리스트 화면은 **전체 검색 / 관리용**
- 수정 / 삭제 / 순서 변경 전에 **미리보기 확인**
- 리스트에서는 **중복 확인 / 다중 삭제 / 드래그 이동** 가능
- `Setting.py`가 핵심 파일이므로 없으면 대부분 기능이 동작하지 않음
- 중요한 작업 전에는 백업하는 습관이 좋음

---

**번호로 직접 수정/삭제/순서 변경을 하거나, 전체 곡 목록을 검색하고 중복을 확인하면서 편집·삭제·이동까지 할 수 있는 노래 관리 도구**입니다.
