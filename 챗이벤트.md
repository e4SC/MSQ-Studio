# 💬 챗 이벤트 관리 창 사용법 (초보자용)

이 창은 **인게임 채팅 정답(별칭/동의어) 목록을 표로 관리하고**, 최종적으로 **`chatevent.txt` 파일을 자동 생성**하는 도구입니다.  
처음 사용하는 사용자도 그대로 따라 할 수 있게 **기능 중심**으로 정리했습니다.

---

## 1) 이 창에서 할 수 있는 것

- 채팅으로 인정될 **정답 텍스트** 를 표 형태로 입력/정리
- 한 줄(한 그룹) 안의 여러 텍스트를 **같은 인덱스(동일 정답 그룹)** 로 묶기
- 전체 중복 방지(같은 정답 텍스트를 여러 곳에 쓰지 못하게)
- 글자수 제한(인게임 안정성을 위한 제한)
- 검색(하이라이트 + 다음 결과로 이동)
- 백업 파일 불러오기 / 자동 백업 생성
- 최종 결과를 **`chatevent.txt`로 생성**

---

## 2) 화면 구성

### 상단 바 (검색/도움)
- 검색 입력칸
- 검색 버튼
- FAQ 버튼
- 도움말 버튼(웹 링크 열기)
- 검색 결과 표시(몇 개 매칭됐는지)

### 중앙 (표)
- 정답 표(여러 열)
- 여러 셀 선택 가능
- 복사/붙여넣기 지원
- 삭제(선택 셀 비우기) 지원

### 하단 바 (백업/열 관리/생성)
- 백업 파일 선택 목록
- 백업 불러오기 버튼
- “추가정답 열 + / -” 버튼(열 늘리기/줄이기)
- `chatevent.txt` 생성 버튼

---

## 3) 기본 개념: “행(줄) = 같은 정답 그룹”

- **한 행(가로줄)** 은 “서로 같은 정답으로 취급될 텍스트 묶음”입니다.
- 예를 들어 아래처럼 입력하면:
  - `아이유`, `IU`, `이지은` 을 **모두 같은 정답**으로 인정하게 할 수 있습니다.

✅ 예시(한 행에 같은 의미의 텍스트를 나란히 입력)
- 첫 칸: `아이유`
- 다음 칸: `IU`
- 다음 칸: `이지은`

> 즉, **행 단위로 묶어서 관리**하는 방식입니다.

---

## 4) 입력 규칙 (중요)

### ✅ 글자수 제한
- 한 칸(셀)에 입력 가능한 글자수는 **최대 26자**입니다.
- 26자를 넘으면 경고가 뜨고 입력이 취소됩니다.

### ✅ 전체 중복 금지
- 표 전체에서 **같은 텍스트를 두 번 이상 사용할 수 없습니다.**
- 이미 다른 곳에 쓰인 텍스트를 다시 입력하면 경고가 뜨고 입력이 취소됩니다.

> 이 두 가지 규칙 덕분에 “인게임에서 꼬이거나 충돌하는 문제”를 예방합니다.

---

## 5) 검색 사용법

### 검색 동작
- 검색어를 입력하면 **일치하는 셀이 색으로 강조(하이라이트)** 됩니다.
- 공백은 무시하고 비교합니다(검색어/셀 모두 공백 제거).

### 다음 결과로 이동
- `Enter` 또는 `F3` → 다음 검색 결과로 이동
- 검색 결과가 없으면 안내창이 뜹니다.

### 빠른 검색 단축키
- `Ctrl + F` → 검색칸으로 이동 + 전체 선택

### 검색 해제
- 검색칸에서 `Esc` → 검색어 지우기

---

## 6) 복사 / 붙여넣기 / 삭제

### 복사
- 셀을 선택 후 `Ctrl + C`

### 붙여넣기
- `Ctrl + V`

붙여넣기는 두 가지 방식이 있습니다:

#### (1) 행렬 붙여넣기(엑셀처럼)
- 탭(`Tab`)과 줄바꿈이 포함된 데이터를 붙여넣으면
- 현재 선택된 셀을 시작점으로 **행/열 형태로 채워집니다.**

#### (2) 단일 텍스트를 여러 셀에 한 번에 붙여넣기
- 여러 셀을 선택한 상태에서
- 클립보드 내용이 “한 줄 텍스트”라면
- 선택된 모든 셀에 동일하게 입력됩니다.

### 삭제(선택 셀 비우기)
- 셀 선택 후 `Del`

### 우클릭 메뉴(컨텍스트 메뉴)
- 복사 / 붙여넣기 / 선택 셀 비우기 기능을 마우스로도 할 수 있습니다.

---

## 7) 열(컬럼) 늘리기 / 줄이기

### “추가정답 열 +”
- 더 많은 별칭 칸이 필요할 때 열을 추가합니다.

### “추가정답 열 -”
- 마지막에 추가한 열을 삭제합니다.

주의:
- 기본 열(최초 구성 열)은 삭제할 수 없습니다.
- 삭제하려는 열에 데이터가 있으면 “정말 삭제할지” 확인을 묻습니다.

---

## 8) 백업 불러오기

### 백업 목록
- 하단의 드롭다운에서 백업 파일을 선택합니다.
- 목록이 비어 있으면 “없음”으로 표시됩니다.

### 백업 불러오기 버튼
- 선택한 백업 파일을 표에 로드합니다.

불러오기 전, 현재 표에 내용이 있으면:
- “현재 내용을 백업한 뒤 불러올지” 확인 메시지가 뜹니다.
- “예”를 누르면 현재 내용을 자동 백업 후 불러옵니다.

---

## 9) `chatevent.txt` 생성하기 (가장 중요)

### 생성 방법
1. 표를 원하는 대로 작성합니다.
2. 하단의 **`chatevent.txt 생성`** 버튼을 누릅니다.
3. 프로그램 폴더에 `chatevent.txt`가 생성됩니다.

### 생성되는 내용(요약)
- 파일에는 기본 헤더 정보가 먼저 들어가고,
- 그 아래에 표의 내용이 **`텍스트 : 인덱스`** 형태로 쭉 기록됩니다.
- 같은 행에 있는 텍스트들은 **같은 인덱스 그룹**으로 묶여 출력됩니다.

### 빠른 생성 단축키
- `Ctrl + S` → 바로 생성 실행

---

## 10) 자동 임시 저장(닫을 때)

창을 닫으면 현재 작성 중인 표가 **임시 파일로 저장**됩니다.  
그래서 실수로 닫아도 다음에 다시 열었을 때 이어서 작업할 수 있습니다.

---

## 11) 추천 작업 흐름(처음 사용자용)

1. 검색으로 기존 항목이 있는지 확인 (`Ctrl + F`)
2. 필요한 정답들을 행 단위로 입력(같은 의미는 같은 행)
3. 중복/글자수 경고가 뜨면 규칙에 맞게 수정
4. 필요하면 열을 추가해서 별칭 칸을 늘림
5. 최종적으로 `chatevent.txt 생성` 실행
6. 인게임/디버그 창에서 채팅 표시가 정상 매핑되는지 확인

---

## 12) 단축키 요약

- `Ctrl + F` : 검색칸 포커스
- `Enter` / `F3` : 다음 검색 결과로 이동
- `Ctrl + S` : `chatevent.txt` 생성
- `Ctrl + C` : 복사
- `Ctrl + V` : 붙여넣기
- `Del` : 선택 셀 비우기
- `Esc` : 검색어 지우기(검색칸에서)

---

## 13) 한 줄 요약

표에 **채팅 정답(별칭)을 행 단위로 묶어 입력**하고, 버튼(또는 `Ctrl+S`) 한 번으로 **`chatevent.txt`를 자동 생성**하는 관리 창입니다.
