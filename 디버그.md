# 🛠️ 정답 디버그 창 사용법

이 문서는 **정답/힌트/객관식/주관식 정보를 한 화면에서 확인하고, 필요하면 바로 수정하는 “디버그 창”**의 사용법을 설명합니다.  
처음 사용하는 사용자도 그대로 따라 할 수 있도록 **기능 중심**으로 정리했습니다.

---

## 1) 실행 전 준비

### ✅ 반드시 필요한 파일
- `Setting.py`  
  - 곡 데이터(제목/주제/정답/힌트/문제/보기 등)가 들어있는 파일입니다.
  - 이 파일이 없으면 디버그 창이 열리지 않습니다.

### ✅ 음원 재생을 하려면 필요한 폴더(권장)
- `sound/mp3/` 또는 `sound/wav/`  
  - 선택한 곡 번호에 맞는 파일이 있어야 재생됩니다.
  - 파일명은 **숫자 번호 기반**이어야 합니다.
  - 예: 1번 곡 → `1.mp3` 또는 `1.wav`

### ✅ 채팅 표시(챗이벤트) 미리보기 정확도를 높이려면(권장)
- `chatevent.txt`
  - “표시할 글자”와 “인덱스 번호”를 연결한 목록 파일입니다.
  - 형식(한 줄에 하나):
    ```
    표시텍스트:인덱스
    ```
  - 예:
    ```
    아이유:4012
    IU:4012
    ```

---

## 2) 화면 구성 이해하기

디버그 창은 크게 **왼쪽(곡 선택/재생)** + **오른쪽(정보 표시/수정)** 으로 구성됩니다.

### 왼쪽: 곡 선택 / 재생
- **포맷 선택**: mp3 또는 wav
- **재생/멈춤 버튼**
- **재생 위치 이동(슬라이더)**
- **볼륨 조절**
- **음악 폴더 열기**
- **검색창 + 곡 목록**
  - 곡 목록에서 곡을 선택하면 오른쪽 정보가 즉시 바뀝니다.

### 오른쪽: 정보 확인 / 미리보기 / 수정
- **안내 영역**: 현재 표시되는 값이 “인게임 기준”이라는 설명
- **채팅 표시 미리보기**: 인게임에서 보일 텍스트 형태를 미리 확인
- **기본 정보**: 제목/주제/노래 + 힌트 + 2차 힌트
- **가수 정보**: 가수 1~7 + 각 힌트 + 각 2차 힌트
- **주관식**: 문제 + 정답들 + 정답 힌트 + 2차 힌트
- **객관식**: 문제 + 보기 1~4 + 정답 번호

---

## 3) 기본 사용 순서 (처음 사용자용)

### ① 곡 찾기
1. 왼쪽 검색창에 키워드를 입력합니다.
2. 아래 곡 목록이 자동으로 필터됩니다.
3. 원하는 곡을 클릭합니다.

### ② 정보 확인
- 오른쪽에 아래 정보가 한 번에 표시됩니다.
  - 제목/주제/노래
  - 힌트/2차힌트
  - 가수/주관식/객관식 내용
  - 채팅 표시 미리보기

### ③ 음원 재생(선택)
1. 왼쪽에서 mp3 또는 wav를 선택합니다.
2. 재생 버튼을 누릅니다.
3. 슬라이더로 재생 위치를 이동할 수 있습니다.
4. 볼륨을 조절할 수 있습니다.

> 음원 파일이 없으면 “해당 번호의 음원이 없음” 같은 안내가 표시됩니다.

---

## 4) 값 수정 방법 (더블클릭 편집)

이 디버그 창의 핵심 기능은 **텍스트를 바로 수정하고 즉시 저장되는 것**입니다.

### 수정 절차
1. 오른쪽에서 수정하고 싶은 항목을 **더블클릭**합니다.
2. 입력창이 뜨면 내용을 변경합니다.
3. 아래 중 하나로 저장됩니다.
   - `Enter` 키를 누른다
   - 다른 곳을 클릭해서 포커스를 옮긴다
4. 저장되면 화면이 자동으로 갱신됩니다.

### 취소
- 수정 중 `Esc`를 누르면 저장하지 않고 취소됩니다.

> 수정 결과는 곡 데이터 파일(Setting.py)에 반영됩니다.

---

## 5) “어떤 것”을 수정할 수 있나요?

오른쪽의 대부분 텍스트는 수정 가능합니다. 대표적으로:

### 기본 정보
- 제목, 주제, 노래 이름
- 제목 힌트/노래 힌트
- 제목 2차 힌트/노래 2차 힌트

### 가수
- 가수 1~7 이름
- 각 가수의 힌트
- 각 가수의 2차 힌트

### 주관식
- 주관식 문제
- 주관식 정답(여러 개)
- 주관식 정답 힌트
- 주관식 2차 힌트

### 객관식
- 객관식 문제
- 보기 1~4
- 정답 번호

---

## 6) 채팅 표시(챗이벤트) 미리보기 읽는 법

채팅 표시 미리보기는 **“인게임에서 실제로 보이는 텍스트” 기준으로 표시되는지** 확인하는 기능입니다.

### 정상 표시
- 텍스트가 자연스럽게 표시되면 정상입니다.

### 비정상 표시(매핑 실패)
- 텍스트 대신 **`[인덱스 ####]`** 형태가 보이면,
  - “해당 인덱스를 어떤 텍스트로 표시해야 하는지 목록에 없어서” 발생할 수 있습니다.
  - 이 경우 `chatevent.txt`에 매핑을 추가하면 개선됩니다.

---

## 7) 단축키 모음

- `Esc` : 창 닫기
- `Ctrl + F` : 검색창으로 이동 + 전체 선택
- 곡 목록/검색창에서:
  - `↑ / ↓` : 곡 이동
  - `Enter` : 다음 곡으로 이동(끝이면 처음으로)

---

## 8) 자주 발생하는 문제 & 해결

### Q1. 음원이 재생되지 않아요
- mp3/wav 폴더가 존재하는지 확인하세요.
- 파일명이 **곡 번호와 일치**하는지 확인하세요.
  - 예: 7번 곡이면 `7.mp3` 또는 `7.wav`

---

### Q2. 수정했는데 반영이 안 된 것 같아요
- 더블클릭 후 입력 → `Enter` 또는 다른 곳 클릭으로 저장됩니다.
- 저장 후 화면이 자동 갱신됩니다.

---

## 9) 한 줄 요약

- **왼쪽에서 곡을 고르고**,  
- **오른쪽에서 정답/힌트/문제를 확인하고**,  
- **더블클릭으로 바로 수정 후 저장**,  
- 필요하면 **음원도 즉시 재생**할 수 있는 디버그 창입니다.
