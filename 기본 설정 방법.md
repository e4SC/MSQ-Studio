# EUD Editor3 기본 설정 방법 (노래 맞히기)

이 문서는 **노래 맞히기 맵을 실행하기 위한 EUD Editor3 기본 설정 방법**을 설명합니다.  
EUD Editor3 **처음 사용하는 유저**와 **기존 사용 유저**를 구분하여 안내합니다.

---

## 🔰 EUD Editor3 처음 사용하는 유저

### 1. EUD Editor3 실행

- 다운 받은 폴더 안의 **EUD Editor3 폴더**를 엽니다.
- 또는 `노래맞히기 (최신버전)` 파일을 실행한 뒤  
  **연결 프로그램을 EUD Editor3로 설정**합니다.

### 5. 맵 파일 경로 설정

사진에 표시된 버튼을 각각 눌러  
아래 항목들을 설정합니다.

- **연결된 맵**
  - 다운받은 폴더 안의  
    `음악퀴즈 맵파일.scx`

- **저장 맵**
  - 실제로 스타크래프트에서 실행할 맵 파일
  - ⚠️ `음악퀴즈 맵파일.scx`와 **같은 이름으로 설정하면 안 됩니다**

### 6. 기본 설정

사진에 표시된 **기본 설정 버튼**을 누른 뒤  
아래 항목들을 설정합니다.

<img width="743" height="669" alt="Image" src="https://github.com/user-attachments/assets/de177dc6-89f0-4c74-900d-cb9e4c85a1a9" />

- **스타크래프트 실행 파일**
  - `StarCraft Launcher.exe`
  - 배틀넷 설정에서 경로 확인 가능
 
<img width="647" height="499" alt="Image" src="https://github.com/user-attachments/assets/a65be4bb-29ae-4432-8838-d3db4dbc738a" />

- **euddraft 실행 파일 (중요)**
  - 다운받은 폴더 안의  
    `euddraft/euddraft.exe`
  - ⚠️ 반드시 해당 파일을 사용해야 합니다. `(버전은 0.9.9.9)`

- **언어**: 한국어  
- **그래픽**: 리마스터  
- **확장자 연결**: 선택하지 않음

---

### 7. 설정 완료

기본 설정은 여기까지입니다.

## 🧑‍🔧 EUD Editor3 기존 사용 유저

기존에 EUD Editor3를 사용해본 적이 있다면  
아래 항목만 확인하면 됩니다.

1. 맵 파일 경로를 현재 프로젝트에 맞게 수정
2. **euddraft 실행 파일**
   - 다운받은 폴더 안의  
     `euddraft/euddraft.exe`
   - ⚠️ 반드시 해당 파일 사용
3. 완료
